| Format A                  | Format B    |
| 011 0000                  | 0111 000    |
| ------------------------- | ----------- |
| e = 011 = 3               | 0111 = 7    |
| E = e - Bias = 3 - 3 = 0  | 7 - 7 = 0   |
| f = 0000 = 0              | 000 = 0     |
| M = 1 + f = 1 + 0 = 1     | 1 + 0 = 1   |
| V = 2^E * M = 2^0 * 1 = 1 | 2^0 * 1 = 1 |


| Format A                  | Format B    |

| e = 101 = 5                      | 1001 = 9              |

| M = 1 + f = 1 + 7/8 = 15/8       | M = 1 + 7/8 = 15/8    |
| V = 2^E * M = 2^2 * 15/8 =  15/2 | V = 2^2 * 15/8 = 15/2 |
